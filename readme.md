 LARANJAL FOODS — Simulador de concorrência 
 Desenvolvido para a cadeira de sistemas operacionais - UFPEL

 - Gera restaurantes, cada um com os recursos: chaves e lanche.
 - Veteranos: pegam primeiro a moto, depois o lanche.
 - Novatos: pegam primeiro o lanche, depois a moto.
 - Modo deadlock: pode ocorrer deadlock quando veterano e novato escolhem o mesmo restaurante ao mesmo tempo e cada um bloqueia um recurso diferente.
 - Modo correção: usa trylock para evitar deadlock.
